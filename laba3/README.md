-  из простого просто скачать 2 файла которые yml, завтем прописать:
docker-compose up -d

# - дальше все это надо настраивать ручками 

# 1 Grafna 


 # 1.1 
- Заходим на http://localhost:3000 (grafna)
- На панельке слева Connections-> Data sourses
- В верхнем правом углу жмем Add new data source
- выбираем prometheus

# 1.2 настрройки подключения к prometheus

- в пункте Connection пишем: http://prometheus:9090
- остальное не трогаем и идем в самый низ где жмем safe&test, должна появиться зеленая плашка что все хорошо

# 1.3 настройка дэшборда

- в панельке слева переходим во вкладку dashboards
- В верхнем правом углу жмем New -> new dashboard
- далее тыкаем import dashboards
- дальше либо выбираем сами на сайте графны(ссылка там есть), либо можно взять например такой 14584
- после чего идем самым просты путем и редачим каждый из показателей(если навестись то в правом верхнем углу будет кнопочка с 3 точкам, и там пункт эдит, туда и заходим)
- во вкладке Datasourse выбираем наш prometheus(будет с зеленой плашкой default)
- далее в строке metric browser ищем job="чему то там", в кавычки вставляем prometheus и жмем Run queries
- если пункта job нету то просто скипаем этот блок(тоесть выходим и удаляем его нафиг) по итогу у нас останется пул рабочих метрик.
